## Requirements

- VSCode 1.74.0 或更高版本

## Development

### 安装依赖

```bash
pnpm install
```

### 编译

```bash
pnpm run compile
```

### 监听模式（自动编译）

```bash
pnpm run watch
```

### 运行测试

```bash
pnpm test
```

### 打包插件

```bash
vsce package
```
